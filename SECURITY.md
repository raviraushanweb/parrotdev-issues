# Security Policy

## Reporting a Vulnerability

Security vulnerabilities **must not** be reported via public GitHub issues.

Please send details to **security@parrotdev.com** instead.

We will acknowledge receipt within 48 hours and work with you to understand and resolve the issue promptly. Please do not disclose the vulnerability publicly until we've had a chance to address it.

## Supported Versions

Only the latest published version of each extension receives security updates.
