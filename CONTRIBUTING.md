# Contributing

Thanks for helping improve ParrotDev!

This repository is for **bug reports and feature requests only**. We don't accept code contributions here — the extensions themselves are closed-source.

## Before opening an issue

- **Search** the existing issues to see if someone already reported it.
- **Check the website** ([parrotdev.com](https://parrotdev.com)) for documentation and FAQs.
- **Update** to the latest extension version and confirm the issue still exists.

## Writing a good bug report

1. Use the **Bug Report** template — it asks for everything we need.
2. Include your **OS, VS Code version, and extension version**.
3. Provide **clear steps to reproduce**.
4. Paste any **relevant logs or terminal output**.

## Writing a good feature request

1. Use the **Feature Request** template.
2. Describe **what problem you're trying to solve** (not just what you want built).
3. Explain **how you envision it working**.

## After submitting

- A maintainer will triage your issue and apply labels.
- We may ask follow-up questions. Please respond promptly so we can help.
